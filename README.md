## 一句话说明

用开源 CRM「**Twenty**」给外贸业务搭一套客户管理系统，本地 Docker 部署，已接入网易企业邮箱做客户邮件归档，未来计划与公司官网 AI 客服打通实现自动获客。

## ⚠️ 三条最重要的提醒（务必记住）

1. **不要随便断开/重连邮箱账户** —— 会触发网易限流，导致同步失败。现在正常，别碰它。
2. **改任何配置或数据前先备份** —— 用 `docs/04` 里的备份命令，数据无价。
3. **这是本地试用环境，不是正式生产环境** —— 电脑关机/休眠服务就停了。要长期稳定用、要对接官网，必须部署到 Linux 云服务器。
